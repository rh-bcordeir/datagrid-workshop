# DataGrid Workshop

This repository contains multiple implementations of the DataGrid Workshop application using different frameworks.

## Projects

### datagrid-quarkus
A Quarkus-based implementation of the DataGrid Workshop application.

For more information, see [datagrid-quarkus/README.md](datagrid-quarkus/README.md).

### datagrid-spring
A Spring Boot-based implementation of the DataGrid Workshop application.

For more information, see [datagrid-spring/README.md](datagrid-spring/README.md).
